## 🚀 Quick Start

Deploy your own AI brain in 3 simple steps:

### 1. Set Up Accounts (10 minutes)
- [Hetzner Cloud](https://hetzner.com/cloud) - Server hosting (€23.46/month)
- [GitHub](https://github.com) - Repository access (free)
- [Hugging Face](https://huggingface.co) - AI models (free)

### 2. Configure Environment (2 minutes)
```bash
# Copy configuration template
wget https://raw.githubusercontent.com/brian95240/ai-apex-brain/main/apex-brain.env.example
cp apex-brain.env.example apex-brain.env

# Edit with your API tokens and passwords
nano apex-brain.env
```

### 3. Deploy System (10 minutes)
```bash
# One-command deployment
curl -fsSL https://raw.githubusercontent.com/brian95240/ai-apex-brain/main/deploy-apex-brain.sh | bash
```

That's it! Your AI brain will be accessible at your server IP.
